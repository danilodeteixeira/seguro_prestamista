## 🚀 **Instruções de Deploy**

### **Passo 1: Upload no Netlify**
1. Acesse https://netlify.com
2. Clique em "Add new site"
3. Selecione "Deploy manually"
4. **Arraste TODOS os arquivos desta pasta**
5. Clique em "Deploy site"
